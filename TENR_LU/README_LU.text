The main function:
[Tole,l,Iter]=tnr_testlu(datain,step);

Tole= tolerance;
l= Lambda (or maximum loadability in this case).
Iter= Number of Newton-Raphson iterations.

datain=IEEE cases. i.e. case9,case14...etc.

step= Newton step size...

case9, alpha=1;
case14, alpha=1;
case30, alpha=1;
case_IEEE30, alpha=1;
case57, alpha=1;
case118, alpha=0.9;
case300, alpha=0.7; (For 300 bus case the maximum tolerance was up to only two digit)
case2383wp, alpha=0.2;
case3012wp, alpha=0.2;
