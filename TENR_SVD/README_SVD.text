The main function:
[Tole,l,Iter]=tnr_testlu(datain,step);

Tole= tolerance;
l= Lambda (or maximum loadability in this case).
Iter= Number of Newton-Raphson iterations.

datain=IEEE cases. i.e. case9,case14...etc.

step= Newton step size...

case9, alpha=0.37;
case14, alpha=0.8;
case30, alpha=0.37;
case_IEEE30, alpha=0.83;
case57, alpha=1;
case118, alpha=0.49;
case300, alpha=0.45
case2383wp, alpha=0.63;
case3012wp, alpha=0.57;
